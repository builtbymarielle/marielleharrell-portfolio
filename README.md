# Marielle Harrell - Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing my work as a Software Engineer.

## 🚀 Features

- **Modern React Architecture**: Built with React 18 and Vite for fast development and optimal performance
- **Responsive Design**: Fully responsive across all device sizes
- **Smooth Animations**: Custom scroll-based parallax effects and fade transitions
- **Project Showcase**: Dynamic project cards with technology icons
- **Optimized Assets**: All images, fonts, and icons properly organized
- **Clean Code**: Well-structured components with hooks for state management

## 🛠️ Technologies Used

- **Frontend**: React 18, JavaScript ES6+
- **Build Tool**: Vite
- **Styling**: CSS3 with custom properties and animations
- **Icons**: Font Awesome
- **Fonts**: Custom Sora and Inter font families

## 📦 Installation & Setup

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```
   This will start the development server at `http://localhost:3000`

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Preview production build**:
   ```bash
   npm run preview
   ```
